
# React + Tailwind Starter

Vite + React + Tailwind, tuned for AI-assisted prototyping.

## Quick Start
1. Install deps: `npm install`
2. Run dev server: `npm run dev`
3. Open the URL printed in your terminal (usually http://localhost:5173).
